┌─ CUSTOMIZATION ───────────────────────────────────────────────────────────┐
│                                                                            │
│ To change dark mode colors:                                               │
│                                                                            │
│ 1. Open: app/globals.css                                                  │
│ 2. Find the .dark { ... } section                                         │
│ 3. Update CSS variables:                                                  │
│                                                                            │
│    .dark {                                                                │
│      --background: #1A1F2E;    /* Main area background */                 │
│      --sidebar: #0F172A;       /* Sidebar background */                   │
│      --accent: #14B8A6;        /* Teal accent color */                    │
│      /* ... other variables ... */                                        │
│    }                                                                       │
│                                                                            │
│ 4. Save the file - changes apply instantly!                               │
